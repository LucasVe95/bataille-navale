# bataille-navale
Projet de bataille navale:
.les instructions de installation:
- installer le logiciel visual studio code 
- installer l extension Python dans le logiciel visual studio code 
- ouvrir le fichier plateau.py avec le logiciel installé comme mentionner dans la premiére partie  
- exécuter le code, la partie peut commencer

.Explication des régles :
chaque joueurs choisis l emplacement  de ses bateaux au début de la partie ,
ensuite la partie peut commencer:
- le O représente un tir raté
- le B représente l emplacement d'un bateau
- le X représente un bateau touché ou coulé
si un des joueurs a coulé tout les bateaux de son adversaire ,il remporte la partie
